# Vest - AI美股交易记录助手

## 功能

帮助你记录美股交易，通过对话完成：
- 📝 投前：记录买入逻辑
- 📸 投中：识别截图入场
- 📊 投后：复盘总结

## 使用方法

### 1. 创建交易计划
发送：`买入 $NVDA，目标价200，止损180`
或：`关注 $TSLA`

AI 会引导你填写买入逻辑。

### 2. 执行入场
发送截图或：`$NVDA 170元买入 100股`

### 3. 平仓
发送截图或：`$NVDA 180元清仓`

### 4. 查看持仓
发送：`持仓`

### 5. 统计
发送：`统计`

## 数据库

- 交易表：trades
- 复盘表：journals
- 状态：PLANNING → OPEN → CLOSED

## API

- POST /api/trades - 创建交易
- POST /api/trades/execute - 入场
- POST /api/trades/close - 平仓
- GET /api/trades - 列表
- GET /api/stats - 统计

## 对话示例

```
用户：想买NVDA
助手：请提供买入逻辑、目标价和止损价。例如：买入NVDA，逻辑是AI芯片需求强劲，目标200，止损180

用户：AI芯片龙头，长期看好，目标200，止损180
助手：✅ 已创建交易计划 $NVDA LONG，目标价$200，止损$180

用户：170元成交100股
助手：✅ 已入场 $NVDA，170元/股 × 100股 = $17,000

用户：190元清仓
助手：✅ 已平仓 $NVIDEA，盈利 $2,000 (盈亏比 +11.76%)
      请简述卖出原因...
```

## 注意事项

- 股票代码带$或不带均可
- 支持截图识别（富途/盈透）
- 数据存储在本地 SQLite
